Setup Instructions (Windows)

  1. Extract the .tar.gz to D:\TerraLPG
  2. Open terminal in that folder
  3. Run npm install
  4. Run npx prisma db push
  5. Run npm run db:seed
  6. Run npm run dev
  7. Open http://localhost:3005
  8. Login: admin@terralpg.ng / demo123
